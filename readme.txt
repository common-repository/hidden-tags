Hide certain tags/categories from the public

== Description ==

Specify a list of tags or categories to keep hidden: These will no longer appear anywhere
on the site, such as in the tag cloud or on the list of post tags.  This is useful when using
tags to control behaviour of your blog with other software, or when you wish to maintain
groupings of posts out of the public eye.

== Installation ==

1. Unzip the package, and upload `hidden-tags` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit 'Hidden Tags' settings underneath WordPress settings to configure
